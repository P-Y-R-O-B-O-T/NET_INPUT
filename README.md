## INSTALLATION
* `cd NET_INPUT`

* `chmod +x install.sh`

* `./install.sh`

## USAGE
* Start the server on the machine that you want to control
```
netinp_server
```
* Start the client on the machine that you want to use to control the other machine
```
netinp_client
```
